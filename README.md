# Wedding Bingo 🎉

🚧 Coming Soon 🚧

This is a personal project I’m currently building for my upcoming wedding.

The idea came from thinking about how to make the time during dinner more fun and interactive for the guests. Instead of just sitting and waiting between courses, guests will be able to participate in a **Bingo-style game** by completing fun tasks and uploading photo proof.

The goal is to create a light, social experience that encourages guests to interact, move around, and capture fun moments during the event.

## Why I Built This

Beyond the wedding itself, the main motivation behind this project was **learning new backend and DevOps technologies** that I hadn't had the chance to use before.

I wanted a project where I could:
- design a small real system
- build the backend architecture from scratch
- work with technologies that are common in production environments

## Tech Stack
